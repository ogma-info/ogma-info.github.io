## 🎣 Qu'est-ce que le Phishing ?

Le phishing (ou hameçonnage) est une technique utilisée par des cybercriminels pour vous inciter à divulguer des informations personnelles (mots de passe, numéros de carte bancaire) en se faisant passer pour une entité de confiance (banque, impôts, Netflix, etc.).

### 🚩 Comment repérer un mail suspect ?

1. **L'adresse de l'expéditeur** : Regardez bien après le @. Si c'est `impots-gouv@gmail.com` au lieu de `...@dgfip.finances.gouv.fr`, c'est un faux !
2. **L'urgence ou la menace** : "Votre compte va être supprimé", "Amende impayée". Ils veulent vous faire paniquer.
3. **Les fautes d'orthographe** : Souvent présents, même si les pirates s'améliorent.
4. **Le lien** : Sur ordinateur, survolez le bouton sans cliquer pour voir l'adresse réelle s'afficher en bas de votre navigateur.

### 🛡️ Comment se protéger ?
- **Ne cliquez jamais** sur les liens dans les mails suspects. Allez directement sur le site officiel via vos favoris.
- **Utilisez Bitwarden** : Il ne remplira jamais vos identifiants sur un faux site !
- **Activez la 2FA** : Même si le pirate a votre mot de passe, il sera bloqué par le code de sécurité.
